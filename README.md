# GTU CSE Study Material Portal

A complete, modern, responsive website for GTU Computer Science Engineering students.

## Features
- **Centralized Study Material**: Notes, Books, PYQs, Syllabus.
- **Semester-wise Organization**: Easy access for Sem 1 to 8.
- **Google Drive Integration**: Admin panel to add Drive links directly.
- **Admin Panel**: Secure dashboard to manage content.
- **Search & Filter**: Find materials by subject, semester, or type.
- **Responsive Design**: Works on Mobile, Tablet, and Desktop.
- **Academic Theme**: Professional White + Blue design.

## How to Run
This is a static web application. You do not need any backend server.

1. Open the folder `gtu-cse-portal`.
2. Double-click on `index.html` to open it in your browser.

## Technology Stack
- **Frontend**: HTML5, CSS3 (Variables, Flexbox/Grid), JavaScript (ES6+).
- **Icons**: Ionicons (CDN).
- **Data**: LocalStorage (Simulated Database). 
  *Note: In a production environment, this would be replaced by a real backend database.*
